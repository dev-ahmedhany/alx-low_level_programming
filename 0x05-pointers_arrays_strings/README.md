## 0. 98 Battery st

Write a function that takes a pointer to an int as parameter and updates the value it points to to 98.

Prototype: `void reset_to_98(int *n);`

```bash
julien@ubuntu:~/0x05$ cat 0-main.c
#include "main.h"
#include <stdio.h>

/**
 * main - check the code 
 *
 * Return: Always 0.
 */

{

    printf("n=%d\n", n);
```
